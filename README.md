Kult cli is toolkit for configuring and bootstrapping Kult applications.

Have a look at our [docs](https://michael616kriel.github.io/kult-docs/) for more information.

## Installation

yarn:

```
yarn add @kult/cli
```

NPM:

```
npm i @kult/cli
```

## Usage

```bash
kult
```
